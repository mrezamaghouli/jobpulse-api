Authentication

Public job search endpoints require an API key.

X-API-Key: YOUR_API_KEY

Metadata and health endpoints are public and do not require an API key.

Search Jobs

Use /api/jobs/search for ranked job search with filters and pagination.

curl -sS \
  -H "X-API-Key: YOUR_API_KEY" \
  "http://35.192.251.190/api/jobs/search?query=data%20analyst&location=Germany&sort_by=relevance&sort_order=desc&limit=10"
